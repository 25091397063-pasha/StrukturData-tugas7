# 🗂️ Implementasi Queue — Struktur Data Antrian

> **Simulasi sistem antrian berbasis Python** dengan analisis mendalam terhadap waktu pelayanan dan kapasitas petugas.

---

## 📌 Tentang Proyek

Proyek ini mengimplementasikan struktur data **Queue (Antrian)** menggunakan bahasa pemrograman **Python**. Mencakup operasi inti antrian, simulasi dunia nyata, serta eksplorasi bagaimana variabel sistem memengaruhi efisiensi antrian secara keseluruhan.

---

## ⚙️ Fitur Utama

| Fitur | Deskripsi |
|---|---|
| `Enqueue` | Menambahkan elemen ke belakang antrian |
| `Dequeue` | Mengambil elemen dari depan antrian |
| Simulasi Antrian | Mensimulasikan alur antrian secara nyata |
| Analisis Waktu Pelayanan | Mengukur dampak lama pelayanan terhadap sistem |
| Analisis Jumlah Petugas | Mengevaluasi pengaruh jumlah server terhadap antrean |

---

## 🔍 Cakupan Implementasi

```
Queue
├── Operasi Dasar
│   ├── enqueue()   → Tambah ke antrian
│   └── dequeue()   → Ambil dari antrian
├── Simulasi Antrian
│   └── Alur pelanggan secara nyata
└── Analisis Sistem
    ├── Waktu pelayanan
    └── Jumlah petugas
```

---

## 🧠 Konsep yang Dipelajari

- Struktur data **FIFO** *(First In, First Out)*
- Pengaruh **service time** terhadap panjang antrian
- Optimasi sistem dengan penambahan **jumlah server/petugas**
- Pemodelan antrian dalam konteks dunia nyata
